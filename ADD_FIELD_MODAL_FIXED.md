## 🎯 What Each Field Type Does

### Text
- Single line text input
- Good for: Names, titles, short descriptions
- Example: "Enter your name"

### Number
- Numeric input only
- Good for: Quantities, measurements, counts
- Example: "How many copies?"

### Select (Dropdown)
- Dropdown menu with options
- Good for: Single choice from many options
- Example: "Choose color: Red, Blue, Green"

### Radio Buttons
- Visual button selection
- Good for: Single choice, visible options
- Example: "Size: Small, Medium, Large"
- **Supports nested fields!**

### Dimension (Size)
- Width × Height input with presets
- Good for: Size selection with common sizes
- Example: "3×4, 5×8, 6×10, Others"

### File Upload
- File selection and upload
- Good for: Design files, images, documents
- Example: "Upload your design"

### Textarea (Multi-line)
- Multi-line text input
- Good for: Long descriptions, notes, instructions
- Example: "Special instructions"

### Date
- Date picker calendar
- Good for: Delivery dates, deadlines
- Example: "When do you need this?"
